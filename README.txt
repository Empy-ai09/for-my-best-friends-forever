CARA PAKAI
==========

1. Edit link galeri:
   - Buka file script.js
   - Ubah baris: const GALLERY_URL = "https://example.com/galeri-nadya";
   - Ganti dengan link Google Drive / Google Photos kamu, simpan.

2. Coba lokal:
   - Double-click index.html (atau drag ke browser).

3. Upload ke GitHub Pages:
   - Buat repo baru di GitHub, misalnya "birthday-nadya"
   - Upload 3 file: index.html, style.css, script.js
   - Buka Settings repo > Pages > Source = "main branch" / root
   - Tunggu beberapa menit, link akan muncul: https://USERNAME.github.io/birthday-nadya/

Selesai! 🎁
